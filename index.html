사용 방법

수료증을 사용하려면 다음 단계를 따르세요:

  1. 위의 세 가지 스타일 중 원하는 수료증을 선택하여 미리보기를 확인하세요.
  2. 선택한 수료증 페이지에서 마우스 우클릭 → "다른 이름으로 저장"을 선택하여 HTML 파일을 저장하세요.
  3. 저장한 HTML 파일을 텍스트 에디터로 열어 수료자 정보를 수정하세요.
  4. 브라우저에서 파일을 열고 Ctrl+P (또는 Cmd+P)를 눌러 인쇄하거나 PDF로 저장하세요.
  5. 인쇄 시 "배경 그래픽" 옵션을 체크하여 모든 디자인 요소가 포함되도록 하세요.
